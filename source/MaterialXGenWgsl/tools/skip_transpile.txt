# Hand-written genwgsl .wgsl files that must NOT be transpiled from genglsl.
#
# This is the authoritative list of hand-maintained WGSL, used by mxgenwgsl.py for two things:
#   * the lib transpile pass skips any genwgsl/lib/*.wgsl listed here (naga can't express them);
#   * `mxgenwgsl.py --clean` deletes every generated genwgsl *.wgsl EXCEPT the files listed here.
#
# Paths are relative to the repository root (a leading `libraries/` is optional). Blank lines and
# `#` comments are ignored. Node shaders skipped by SKIP_PATTERNS (image/hextiled/light) are also
# listed here so --clean retains them.
